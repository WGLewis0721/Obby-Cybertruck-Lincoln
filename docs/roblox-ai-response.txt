--------------------------------------------------------------------------------
❌ ISSUE: Scripts Embedded in Workspace Objects
--------------------------------------------------------------------------------
LOCATION: Workspace (School, Urban House, Playground, RoadSections)
SEVERITY: Warning
PROBLEM:
	Over 200 scripts are embedded directly in world objects (doors, lights,
	appliances, traffic lights, etc.). This violates Roblox best practices.
	
	Examples:
	- Workspace.School.Doors.Door.Script
	- Workspace.Urban House.Body.Doors.*.InsideDoor.Script
	- Workspace.School.Lights.*.LightSwitchScript
	- Workspace.RoadSections.*.TrafficLightScript

FIX:
	Option A (Recommended): Move to ServerScriptService/WorldHandlers/
	- Use CollectionService tags to identify interactive objects
	- Single script handles all doors, all lights, etc.
	
	Option B: Keep in Workspace but organize:
	- Workspace/WorldScripts/Doors/
	- Workspace/WorldScripts/Lights/

--------------------------------------------------------------------------------
❌ ISSUE: Deprecated BodyGyro Usage
--------------------------------------------------------------------------------
LOCATION: Multiple (A-Chassis, all door scripts)
SEVERITY: Warning
PROBLEM:
	BodyGyro is deprecated. Should use AngularVelocity constraint.
	
	Found in:
	- Workspace.Tesla Cybertruck.A-Chassis Tune.Initialize
	- ServerStorage.Tesla Cybertruck.A-Chassis Tune.Initialize
	- All door scripts in School and Urban House

FIX:
	Replace BodyGyro with AngularVelocity constraint.
	This is especially important for vehicle physics.

--------------------------------------------------------------------------------

LOCATION: ReplicatedStorage

	- ReplicatedStorage/Remotes/
